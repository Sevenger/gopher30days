## １０２·二叉树的层序遍历

题目链接：https://leetcode-cn.com/problems/binary-tree-level-order-traversal/

### 描述

给你一个二叉树，请你返回其按 **层序遍历** 得到的节点值。 （即逐层地，从左到右访问所有节点）。

### 样例

#### 样例１

二叉树：`[3,9,20,null,null,15,7]`,

```markdown
    3
   / \
  9  20
    /  \
   15   7
```

返回其层序遍历结果：

```markdown
[
  [3],
  [9,20],
  [15,7]
]
```
