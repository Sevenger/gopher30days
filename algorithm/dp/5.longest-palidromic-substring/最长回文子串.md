## ５·题目名称

题目链接：https://leetcode-cn.com/problems/longest-palindromic-substring/submissions/

### 描述

给你一个字符串 `s`，找到 `s` 中最长的回文子串。

### 样例

#### 样例１

```markdown
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

#### 样例２

```markdown
输入：s = "cbbd"
输出："bb"
```

#### 样例３

```markdown
输入：s = "a"
输出："a"
```

### 提示

- `1 <= s.length <= 1000`
- `s` 仅由数字和英文字母（大写和/或小写）组成
